KievII by Jane's Conference

KievII - Javascript library for Web audio apps.

KievII is a Javascript library made to help building audio applications for the Web. For now, KievII implements some GUI element (ie labels, knobs, multiknobs, multi-band displays) and uses HTML5 canvas to draw them (but you can extend it to use whatever you want). Graphic elements chaining is supported in a seminal way (i.e. one element can change the value of one or more elements). Audio is work in progress. For now, KievII aims to be compatible with Mozilla Audio API + audiodata.js and to provide some audio algorithm (for example, pitchshifting) on top of dsp.js

Dependencies
